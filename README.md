# Ledgers Dashboard

Interactive financial analytics dashboard built with **Vue 3**, **Vite**, **Tailwind CSS**, **PrimeVue**, and **Chart.js**.

---

## Tech Stack

* **Vue 3** (Composition API)
* **Vite**
* **Tailwind CSS**
* **PrimeVue**
* **PrimeIcons**
* **Chart.js**

---

## Features

* Financial **KPI cards**
* Multiple **data visualizations**

  * Area charts
  * Line charts
  * Bar charts
* **Scenario tuners** with sliders
* **Responsive dashboard layout**
* Modern UI with **Tailwind design tokens**

---

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the development server

```bash
npm run dev
```

The app will run on:

```
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
